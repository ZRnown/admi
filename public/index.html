使用说明

  • 一个转发实例对应一个 Discord Token，可选择机器人 Token 或自用号 Token。
  • 每个转发实例都可以单独配置转发规则、关键词、排除/替换规则。
  • 转发时如需伪装为源用户，请勾选「使用源用户昵称和头像」。
  • 排除关键词会直接删除命中的词，而不会阻止整条消息转发。
  • 可以添加多个转发实例并行运行，后端会为每个实例独立登录并转发。
  • 关于 Token：
  • • 此 Token 与 Discord Bot Token 不同。
  • • 此 Token 在您更改密码时会发生变化。
  • • 如果启用了双因素认证（2FA），需要先禁用它才能获取 Token。
